# PopcornPal
PopcornPal is a dynamic and visually engaging movie application built using React.js,
With intuitive search functionality, interactive movie cards, and captivating text effects, 
The user-friendly interface, responsive design, and innovative features make PopcornPal the go-to choice for personalized movie recommendations and information. 
Built on React.js, PopcornPal ensures seamless rendering and a smooth user interface
